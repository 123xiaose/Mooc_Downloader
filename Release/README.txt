双击 Setup.exe 进行解压安装即可，会自动在桌面创建一个快捷方式。
程序支持中国大学慕课视频以及课件自动下载，自动创建一个文件夹，并将视频，字幕和课件存放至该文件夹下
可用 PotPlayer 播放器打开 “播放列表.dpl” 文件顺序观看所有视频
建议使用在 windows10 操作系统下使用，Windows7 可能部分不支持
有问题欢迎加群 196020837, 博客: https://blog.csdn.net/qq_16166591/article/details/85249743
